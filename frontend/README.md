# Doc Share App - Frontend

Frontend application for the Document Sharing App.

## Tech Stack

* React
* TypeScript
* Vite
* React Router
* Axios

## Features

* Login & Registration UI
* Document Dashboard
* Upload Documents
* Shared Documents View
* Responsive Design

## Project Structure

src/
├── api/
├── pages/
├── components/
├── hooks/
├── types/
└── App.tsx

## Installation

Install dependencies:

```bash
npm install
```

## Environment Variables

Create a `.env` file:

```env
VITE_API_URL=http://localhost:5000
```

## Run Development Server

```bash
npm run dev
```

Application runs on:

http://localhost:5173

## Build Production Version

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
